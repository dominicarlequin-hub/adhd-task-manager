# ADHD Task Manager

A one-thing-at-a-time task manager built for ADHD brains: a single "Right Now" focus task instead of an overwhelming list, an optional "break this into steps" helper, and a quick-add flow where only one field is required.

## Stack
Vite + React + TypeScript + Tailwind CSS + lucide-react icons.

## Running locally
```
npm install
npm run dev
```

## Deploying
Push this to a new GitHub repo, then import it into Vercel (same flow as your other apps — auto-deploys on every push to main).

## Notes
- Tasks persist in `localStorage` (no backend yet).
- Palette: cream / sage / terracotta — matches the Figma design.
